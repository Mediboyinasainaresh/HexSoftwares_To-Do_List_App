 📝 Advanced To-Do App (Android - Kotlin)

A modern To-Do List application built using Kotlin in Android Studio.

🚀 Features
- Add, Edit, Delete tasks
- Mark tasks as completed
- Priority levels (Low, Medium, High)
- Search functionality
- Swipe to delete
- Persistent storage using SharedPreferences

🛠 Tech Stack
- Kotlin
- RecyclerView
- Gson
- Material UI
